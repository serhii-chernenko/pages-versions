# Drizzle Schema

Usually, when a table is created, it contains general fields like `id`, `created_at`, `updated_at`, etc. But in this case, we need to create a table that will contain the version of the page.

::alert{color="info" icon="i-lucide-lightbulb"}
I'm not sure if this is the best way to implement the schema, but I did my best \:D
::

I created 3 tables:
1. `pages` - contains the page data
2. `page_versions` - contains the version data
3. `page_contents` - contains the content of a specific version

## Tables

::code-content{toltip=true}
#path
[server/db/schema.ts](https://github.com/serhii-chernenko/pages-versions/blob/main/server/db/schema.ts){target="_blank"}

#default
```ts
import { sql } from 'drizzle-orm'
import { sqliteTable, text, integer } from 'drizzle-orm/sqlite-core'

export const pages = sqliteTable('pages', {
  id: integer().primaryKey(),
  deletedAt: text('deleted_at'),
})

export const pageVersions = sqliteTable('page_versions', {
  id: integer().primaryKey(),
  pageId: integer('page_id').notNull().references(() => pages.id, { onDelete: 'cascade' }),
  status: text({
    enum: ['draft', 'scheduled', 'published', 'modified', 'archived'],
  }).notNull().default('draft'),
  createdAt: text('created_at')
    .notNull()
    .default(sql`(current_timestamp)`),
})

export const pageVersionContents = sqliteTable('page_content', {
  id: integer().primaryKey(),
  pageId: integer('page_id').notNull().references(() => pages.id, { onDelete: 'cascade' }),
  versionId: integer('version_id').notNull().references(() => pageVersions.id, { onDelete: 'cascade' }),
  title: text().notNull(),
  slug: text().notNull().unique(),
})
```

## Relations

Also we need to create a relation between the tables. The `pages` table will have many versions, and the `page_versions` table will have one page. The `page_version_contents` table will have one version and one page.

::code-content{toltip=true}
#path
[server/db/schema.ts](https://github.com/serhii-chernenko/pages-versions/blob/main/server/db/schema.ts){target="_blank"}

#default
```ts
import { relations } from 'drizzle-orm'

export const pagesRelations = relations(pages, ({ many }) => ({
  versions: many(pageVersions),
}))

export const pageVersionsRelations = relations(pageVersions, ({ one }) => ({
  page: one(pages, {
    fields: [pageVersions.pageId],
    references: [pages.id],
  }),
  content: one(pageVersionContents, {
    fields: [pageVersions.id],
    references: [pageVersionContents.versionId],
  }),
}))

export const pageVersionContentsRelations = relations(
  pageVersionContents,
  ({ one }) => ({
    page: one(pages, {
      fields: [pageVersionContents.pageId],
      references: [pages.id],
    }),
    version: one(pageVersions, {
      fields: [pageVersionContents.versionId],
      references: [pageVersions.id],
    }),
  }),
)
```
