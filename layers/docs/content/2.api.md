# API

1. When the [main listing page](/){target="_blank"} is opened. It calls the api `/api/pages`.
2. It uses the `findMany` method of Drizzle ORM query builder to get the list of pages. 
3. It uses the `where` method to filter the pages that are not archived (emulation of actual deleting). 
4. It uses the `limit` and `offset` methods to paginate the results. 
5. It uses the `columns` method to select only the `id` column.

```ts
await db.query.pages.findMany({
  where: (schema, { isNull }) => isNull(schema.deletedAt),
  limit: parseInt(query.data.limit),
  offset: (parseInt(query.data.page) - 1) * parseInt(query.data.limit),
  columns: {
    id: true,
  },
  with: {
    versions: {
      limit: 1,
      orderBy: (schema, { desc }) => desc(schema.createdAt),
      columns: {
        id: true,
        status: true,
        createdAt: true,
      },
      with: {
        content: {
          columns: {
            title: true,
            slug: true,
          },
        },
      },
    },
  },
})
```
