# Pages versions

This is a working prototype that used in a real project.

::alert{color="info" icon="i-lucide-lightbulb"}
It is not a production-ready solution.
::

## Idea

I'd like to implement a solution of page versions to have an editing history by using [Drizzle ORM](https://orm.drizzle.team){target="_blank"} and [Cloudflare D1 SQLite database](https://developers.cloudflare.com/d1/){target="_blank"}.
