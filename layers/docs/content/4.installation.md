# Local installation

In case if you want to install the application locally, you can do it by following these steps.

## Prerequisites

- [Bun.sh](https://bun.sh/){target="_blank"}
- [Cloudflare account](https://dash.cloudflare.com/){target="_blank"}
- [Cloudflare D1 database](https://developers.cloudflare.com/d1/getting-started/){target="_blank"} requires a credit card in your acconunt even if you are using a free plan. You can use the [free tier](https://www.cloudflare.com/products/d1/){target="_blank"} of D1 database.

## Steps

Clone the repository.
::code-content
```bash
git clone git@github.com:serhii-chernenko/pages-versions.git
```
::

Go to the project directory. Install the dependencies.
::code-content
```bash
cd pages-versions
bun install
```
::

Prepare drizzle migrations.
::code-content
```bash
# Expected output is .drizzle/migrations/*.sql
bun run db:generate
```
::

Create a new D1 database for Cloudflare.
::code-content
```bash
bunx wrangler d1 create pages-versions
```
::

Get the `database_id` from the output and set it in the `.env` file as `DB_ID`.
::code-content
```bash
cp .env.example .env
```
::

::code-content{toltip=true}
#path
[.env](https://github.com/serhii-chernenko/pages-versions/blob/main/.env.example){target="_blank"}

#default
```dotenv
DB_ID="xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
```
::

Build the project to generate a `wrangler.json` file.
::code-content
```bash
# Expected output is .output/server/wrangler.json
bun run build
```
::

Apply the migrations to the D1 database.
::code-content
```bash
# Expected output is .wrangler/state/v3/d1/*.sqlite
bunx wrangler d1 apply DB
```
::

Copy the auto-generated `wrangler.json` file to the root directory. Because it will be removed when `bun run dev` runs.
::code-content
```bash
cp .output/server/wrangler.json wrangler.json
```
::

Generate environment types for the project. Optional but recommended.
::code-content
```bash
bun run cf-typegen
```
::

Start the development server.
::code-content
```bash
bun run dev
```
::

Open [http://localhost:3000](http://localhost:3000){target="_blank"} in your browser.
